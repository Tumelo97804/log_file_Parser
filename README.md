# Windows Event Log Parser

## Description
This Python script parses Windows Event Logs (System and Application) and generates a CSV report of all error events.  

It demonstrates:
- Windows Event Log API usage (`pywin32`)
- Python scripting and CSV report generation
- Automation and system monitoring skills

## Skills Demonstrated
- Python scripting and error handling
- Working with system logs
- Data extraction and report generation
